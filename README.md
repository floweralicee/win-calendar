# Win Calendar

A personal monthly calendar for tracking daily wins. Minimal, printed-paper aesthetic — the calendar itself is the entire UI.

## Stack

React 18, TypeScript, Vite. No other runtime dependencies by design.

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Agent Instructions

All AI coding agents working in this repo must read **[AGENTS.md](./AGENTS.md)** first. It's the single source of truth for architecture, conventions, and what not to do. It self-updates as the project evolves.
